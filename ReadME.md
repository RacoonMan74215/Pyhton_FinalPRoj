#  Анализ рынка криптовалют: Структура, Ликвидность и Волатильность

> **Дисциплина:** Анализ данных на Python  
> **Инструменты:** Python, Pandas, NumPy, Scikit-learn, Matplotlib, Requests  
> **Источник данных:** CoinGecko API v3

##  О проекте

Данный проект представляет собой комплексное исследование рынка криптовалют (Топ-100 активов по капитализации). Цель работы — не просто собрать данные, а выявить скрытые закономерности, оценить риски и провести автоматическую сегментацию активов на основе методов машинного обучения без учителя.

В ходе исследования решаются следующие задачи:
1. Сбор репрезентативной выборки через REST API с реализацией пагинации.
2. Глубокая предобработка данных и создание финансовых признаков (Feature Engineering).
3. Проверка статистических гипотез о связи метрик ликвидности.
4. Анализ временных рядов (расчет волатильности и скользящих средних).
5. Кластеризация рынка методом K-Means для выделения профилей риска.



## Стек технологий

*   **Сбор данных:** `requests` (REST API), работа с JSON, обработка HTTP-ошибок.
*   **Обработка данных:** `pandas`, `numpy` (векторные вычисления, очистка пропусков).
*   **Статистика:** `scipy` (корреляции Пирсона и Спирмена, проверка нормальности).
*   **Машинное обучение:** `scikit-learn` (K-Means, StandardScaler, метод локтя).
*   **Визуализация:** `matplotlib`, `seaborn` (дашборды, тепловые карты, графики с двумя осями Y).
